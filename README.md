# Portfolio Personal - Eduardo Elias

Este es mi portfolio personal creado como proyecto para la asignatura de Lenguaje de Marcas. El objetivo principal es mostrar mis conocimientos en HTML5 y CSS, y presentar de forma ordenada y clara mi información profesional y académica.

## Contenidos del Portfolio

- **Página Principal (index.html):**  
  Contiene el encabezado principal, presentación personal, y navegación hacia las diferentes secciones.

- **Secciones Incluidas:**  
  - Sobre mí: descripción personal y profesional.  
  - Formación: estudios realizados, incluyendo mi curso actual.  
  - Proyectos: ejemplos de trabajos y prácticas realizadas.  
  - Contacto: formulario básico para que me contacten.

## Tecnologías usadas

- HTML5 semántico para una correcta estructuración del contenido.  
- CSS externo para diseño visual, responsividad y efectos interactivos.  
- Git y GitHub para control de versiones y publicación con GitHub Pages.
